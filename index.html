Basic understanding about git

git init: khoi tai git, bat dau su dung git

git status: check file trong repo chua, kiem tra file duoc add chua, mau xanh duoc add roi, mau do thi chua

git commit: the hien message

git branch: chon nhanh de day vao

git remote add origin + dia chi: xac dinh dia chi

git push: day code
